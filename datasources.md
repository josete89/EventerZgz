#SPARQL queries

## Categories

### Query
```sparql
PREFIX tema: <http://vocab.linkeddata.es/kos/cultura-ocio/tema/>
SELECT DISTINCT ?id ?tema WHERE {	
?uriCont a tema:evento-zaragoza;
   skos:notation ?id;
   skos:prefLabel ?tema.		
}
```
